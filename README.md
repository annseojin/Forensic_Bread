# 윈도우 악성 프로그램 탐지 및 분석 연구

### 📌 프로젝트 정의

1. 평소에는 정상적으로 사용되지만, 특정 조건에서 악의적으로 이용될 수 있는 프로그램 탐지 및 분석
2. Windows 환경에서 사용자 행위 기반의 아티팩트 분석 기법을 확립 후 디지털 흔적을 기반으로 악용된 프로그램 아티팩트를 탐지하고 분석하여 포렌식 자동화 툴 제작
